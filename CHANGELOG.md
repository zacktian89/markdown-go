# Changelog

## 1.1.1

- Replaces the mistaken `1.1.0` release with corrected personal-release metadata.
- Keeps the skill installation flow introduced in `1.1.0` unchanged.

## 1.1.0

- Added `markdown-go install-skill` for Codex, Claude Code, Cursor, and Antigravity-compatible skill directories.
- Added packaged skill assets plus PowerShell and shell bootstrap scripts for one-click installation.
- Added release verification for skill installation and improved Windows npm command handling.
- Pinned installed skills to the local CLI entrypoint so real environments can invoke `markdown-go` reliably after installation.

## 1.0.0

- Initial open source preparation for the `markdown-go` brand.
- Renamed the public CLI and package from `md-to-wechat` to `markdown-go`.
- Added repository documentation, license, contributor guide, and CI validation.
- Tightened packaging and repository hygiene for public release.
